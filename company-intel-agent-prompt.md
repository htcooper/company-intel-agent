# Company Intelligence Agent — Claude Code Project Brief

Read my existing Claude Code skills in "C:\Users\holli\OneDrive\Code Projects\Claude projects\PM Skills" before starting. I am taking a competitive intelligence skill that I built for my company Blackthorn and generalizing it so anyone can use it without exposing Blackthorn information.

## What to build

A Streamlit app called **Company Intelligence Agent** that takes a company name as input and produces a 1-page executive intelligence brief. The app runs an agentic AI pipeline — not a single prompt — that autonomously researches the company across multiple dimensions, then synthesizes findings into a structured output.

Deploy target: HuggingFace Spaces (Streamlit SDK, free tier).

## Why this exists

This is a portfolio project. The primary audience is hiring managers and recruiters at AI-native companies who will spend ~30 seconds evaluating it. It must work with zero friction (no API key required to try it).

## Architecture

```
User inputs company name
        │
        ├── Research Pass 1: News & announcements    (Claude + web search)
        ├── Research Pass 2: Hiring signals           (Claude + web search)
        └── Research Pass 3: Content & positioning    (Claude + web search)
                │
                ▼
        Synthesis Call: Cross-reference all 3 passes → structured 1-page brief
                │
                ▼
        Cache result → render as markdown + offer download
```

Each research pass is a separate Anthropic API call using Claude Sonnet with the `web_search_20250305` tool. The synthesis call takes the outputs of all 3 passes and produces the final brief. This multi-step, tool-using, autonomous pipeline is what makes it agentic — not just a wrapper around a single prompt.

## Technical requirements

- **Framework:** Streamlit
- **LLM:** Anthropic API, Claude Sonnet (`claude-sonnet-4-20250514`)
- **Search:** Anthropic's built-in web search tool (no separate search API needed)
- **API key:** Loaded from environment variable `ANTHROPIC_API_KEY`. My key is the default. Optional BYOK field for power users who want unlimited runs.
- **Rate limiting:** Session-based, 3 runs per session (HF Spaces doesn't expose client IPs to Streamlit)
- **Caching:** File-based, 24hr TTL. If someone searches the same company within 24 hours, serve cached result instead of re-running the pipeline.
- **Graceful degradation:** When API key is missing or budget is exhausted, show a pre-cached example output with a note that the live demo is temporarily unavailable. Pre-generate 2-3 real example outputs during testing to use as fallbacks.
- **Download:** Button to download the brief as markdown.
- **Single file:** Keep everything in `app.py` for simplicity. No separate modules.

## Research pass prompts

Each pass should instruct Claude to search the web for specific information about the company. The prompts should:
- Be specific about what to look for (not vague)
- Request structured output (not freeform rambling)
- Focus on the last 6 months of activity

### Pass 1: News & Announcements
Search for recent press releases, product launches, partnerships, funding rounds, acquisitions, and strategic moves.

### Pass 2: Hiring Signals
Search job boards and careers pages for open roles. Identify which teams are growing, what technologies appear repeatedly, and what the hiring patterns signal about future strategy.

### Pass 3: Content & Positioning
Search for blog posts, executive interviews, conference talks, and thought leadership. Identify the public narrative they're building and how they're positioning themselves.

### Synthesis
Takes all 3 pass outputs and produces a brief with these sections:
1. **What they're doing** — concrete actions (2-3 sentences)
2. **What they're saying** — public narrative (2-3 sentences)
3. **What hiring signals suggest** — talent strategy insights (2-3 sentences)
4. **Bottom line** — synthesis of where the company is headed (1-2 sentences)

Every sentence should contain a concrete insight. No generic filler. End with:
*Generated by Company Intelligence Agent — an agentic AI research tool by Hollis Cooper*
